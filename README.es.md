# Proposal Engine (Alternate README)

This file is intentionally kept in English to maintain a single documentation language across the repository.
Use [`README.md`](./README.md) as the primary entry point.

## Summary

Proposal Engine is an internal SaaS tool for **CreditCheck** that builds commercial proposals (White-Label, Leads, and Combo) with:

- Real-time preview
- PDF export
- HTML export
- DOCX export
- Print-ready output

## Start

```bash
npm install
npm run dev
```
